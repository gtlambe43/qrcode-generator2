# qrcode-generator2
Static webpage that generates a QR code from text or URLs and allows PNG download.
Create a static webpage application that generates a QR code from a line of text (such as a URL).
The page should display the QR code and allow the user to download it as a PNG file.
Test the page by entering sample URLs and verifying the QR code output.
The agent should create a static webpage without any special deployment steps, to be served with the default GitHub Pages workflow.
Provide in the report: screenshots of the page, example QR codes generated, and confirmation that the download works.
